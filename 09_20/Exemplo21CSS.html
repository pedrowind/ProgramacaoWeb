O que é o grid layout?

O grid layout é uma técnica para construção de layouts mais complexos, o seu layout é baseado em linhas e colunas, ou seja, ele é bidimensional

Quando comparado ao Flexbox, nós temos algumas diferenças, pois o Flexbox é unidimensional, ordenando os elementos em linha.

Flexgrid ou Grid Layout

O flexbox e o Grid Layout podem ser usados em conjunto. Você pode aplicar o flexbox a elementos dentro de um layout de grade, aproveitando o melhor das duas técnicas.

A escolha entre flexbox e Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu layout e escolher a técnica que melhor as atende.
